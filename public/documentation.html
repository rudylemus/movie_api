Documentation Page

Intro

This API will provide a list of popular movies along with genres, actors, movie ratings, etc.

Organizing Rest API movie data

Description URL HTTP Method Request body data format Response body data format
Return a list of movies to the user. /movies Get none a JSON object holding information on all the movies.
Return data on a single movie to the user by movie title /movies/:Title Get none a JSON object holding information which contains the title, director, image URL, description, and genre of a single movie by title to the user 
                            
                                {
                                    id: '',
                                    title: '',
                                    description: '',
                                    director: '',
                                    genre: '',
                                    image: '',
                                }
Return data about a genre by name /movies/genre/:Name Get none A JSON object holding data about the type of genre: 
                            
                                {
                                    genre: '',
                                    description: ''
                                }
Return data about a director /movies/directors/:Name Get none A JSON object holding data about the director: 
                            
                                {
                                    name: '',
                                    bio: '',
                                    birth:''
                                }
Allow new users to register /users Post A JSON object holding data about the user such as: 
                            
                                {
                                    username: '',
                                    password: '',
                                    email: '',
                                    birthday: '',
                                    favoriteMovies: []
                                }
A JSON object holding data about the users added which includes a unique ID 
                            
                                {
                                    id: '',
                                    username: '',
                                    password: '',
                                    email: '',
                                    birthday: '',
                                    favoriteMovies: []
                                }
Allow users to update their user information /users/:Username Put none 
                            
                                {
                                    id: '',
                                    username: '',
                                    password: '',
                                    email: '',
                                    birthday: '',
                                    favoriteMovies: []
                                }
Allow users to add a movie to list of favorites /users/:username/movies/:MovieID Post none A message confirming that a movie has been added to favorite list.
Allow users to remove a movie from their favorite list /users/:Username/movies/:MovieID Delete none A message showing that a movie has been removed from their favorites
Allow existing users to deregister /users/:Username Delete none A message indicating whether or not deregistration was completed successfully.
