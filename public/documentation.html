Documentation Page

Intro

This API will provide a list of popular movies along with genres, actors, movie ratings, etc.

Organizing Rest API movie data

Description URL HTTP Method Request body data format Response body data format
Return a list of movies to the user. /movies Get none a JSON object holding information on all the movies.
Return data on a single movie to the user by movie title /movies/[name] Get none a JSON object holding information which contains the title, director, image URL, description, and genre of a single movie by title to the user 
                            
                                {
                                    title: '',
                                    description: '',
                                    director: '',
                                    genre: '',
                                    year: '',
                                    image: '',
                                }
Return data about a genre by name /movies/genre/[genreName] Get none A JSON object holding data about the type of genre: 
                            
                                {
                                    name: '',
                                    description: ''
                                }
Return data about a director /movies/directors/[directorsName] Get none A JSON object holding data about the director: 
                            
                                {
                                    name: '',
                                    bio: '',
                                    birth:''
                                }
Allow new users to register /users Post A JSON object holding data about the user such as: 
                            
                                {
                                    name: '',
                                    favoriteMovies: []
                                }
A JSON object holding data about the users added which includes a unique ID 
                            
                                {
                                    id: '',
                                    name: '',
                                    favoriteMovies: []
                                }
Allow users to update their user information /users/[id] Put none 
                            
                                {
                                    id: '',
                                    name: '',
                                    favoriteMovies: []
                                }
Allow users to remove a movie from their favorite list /users/[id]/[movieName] Delete none A message showing that a movie has been removed from their favorites
Allow existing users to deregister /users/[id] Delete none A message indicating whether or not deregistration was complete
